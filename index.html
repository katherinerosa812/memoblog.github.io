Becoming a Project Manager

Becoming a Project Manager takes some time. I recommend starting in techincal support. This way you learn the product very well, which will come in handy later on as a Project Manager.

Next, learn Jira, Smartsheets, and MS Project. These tools are used often as a Project Manager. Secondly, learn agile methodologies, including Scrum, Waterfall, and Kanban.
